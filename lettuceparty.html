Lettuce Party
9 February 2017 ║ View on GitHub

While interning at Zomato, I worked on some illustrations for a Zomato iMessage sticker pack. I wanted a hand-drawn look for the stickers, so I was manually doing the lettering.


The original Lettuce Party sticker.

Since I quite liked how it was coming out, I wanted to try making a font out of it. I looked up softwares for creating fonts, and came across BirdFont — a free, open source font editor.

BirdFont has a lot of weird quirks, but it has some documentation so it wasn’t too hard to play around with it to figure it out. It supports copy-pasting from Illustrator, so I created my glyphs in illustrator, and then pasted them into BirdFont.


The BirdFont interface.
The result

After making the capital letters, I realised the result was better than I had expected, so I decided to complete the default unicode glyphs - uppercase, lowercase, numbers and some punctuation. Here’s what it looks like :


Lettuce Party v0.1

Notes

I wish I’d read up a little more about things I needed to keep in mind before I got started, because I created all my letters a lot smaller than I should have. This means the default line height is way too much. It can be fixed by using custom line heights, but it’s still not ideal, and isn’t trivial to fix.


There doesn’t seem to be a way to set the line height for the font other than adjusting the size of the characters :(
Feedback

If you have any suggestions, you can file an issue on GitHub or write to me at feedback@gyanl.com. You can also check out the BirdFont file on the GitHub repo.
